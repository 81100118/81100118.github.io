四麻升五段、三麻升特纪念

发表于 2019-02-07 分类于 日麻

四麻升五

2019年1月24日晚，SB成功升到五段。都说四升五比三升四还简单，我却花了215战。刚升特的时候R只有1806，打了四把特南就回到了上级，真正的特上一日游。在上级打了十几战之后R打回了1800，但在特上打了两把之后再次贫乳掉回上级。之后在上级pt一直输到410，触底反弹，一波狗到995pt R1809，接着在特上继续狗到了1505/1600pt。此时离五段只差一个1位一个2位，偏偏就开始屎，死到了830pt，差点回四段原点。好在特上R正和不会贫乳。最后一波狗到了五段。值得一提的是还差80pt的时候，我选择了去上东，因为上东一个top40pt，两个top正好升段。如果在上东吃2的话，就去特南，因为还差70pt，特南top75pt正好。
